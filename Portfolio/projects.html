Orchid√©e
Orchidee was a project developed with Symfony. It is a web application that we built for a real customer. Basically, our client wanted an application that had 5 main features: display an employee's schedule, allow to take appointments, have multiple types of accounts, show account information and give her business a web presence. Indeed, she was using only Facebook before.
While working with Symfony, we improved our knowledge about how MVC works. We also learned a new framework from scratch since we had no classes about it. We had to do a lot of research and a lot of effort in order to properly make a web-application with Symfony.

Running Dog
My first game! This final project for my gaming class made me so happy! I finally worked on a game. However, it wasn't as easy as expected. Running dog is a basic side-scroller about a dog trying to escape from his master, however, it isn't easy!
In that class a learned a lot about how we actually create a video game and how to make it fun. Therefore, my friend and I tried our best to put what we learned into practice. Most of the game is hand-made, yes the dog too! We are no professional pixel artists, but we still had fun pretending so.
This game was developed in C# and we used a lot of common things from Unity such as prefabs.

Skills Developed     Unity     C#     Game development

What kind of shop are we?
What kind of shop are we? was a semester-long project. However, the difficulty was that we did not fully create it. Indeed, the code was already given. Easy? Not at all, the code was full of mistakes and we had to correct it in order to make it work. We also had to add our personal touch.
This project was developed with PHP and it implements everything we learned in the course. From variable declaration to how to handle cookies and sessions, everything is in there.

Basic security testing with Kali Linux
As I was searching for a place to start in the security field, I've learned about Kali Linux. I've played a lot with it by myself, but it was obvious that I was going nowhere. Then I bought a book for beginners. "Basic security testing with Kali Linux" taught a lot about a lot. It covers enough material for beginners in multiples domains such as Wifi hacking, password decryption and some of the most used tools in the field such as Nmap and Metasploit. Excellent reading to begin somewhere.

Nmap: Network exploration and security auditing cookbook
Nmap is one of the most used tools in the security filed for reconnaissance. The tool itself has evolved a lot thanks to the many contributors. Not only does it scans networks, but it can also now search for vulnerabilities while doing so! This book showed and explained to me the many commands that Nmap has and are used in order to scan and secure networks. It was a really interesting experience to scan my own computers.
